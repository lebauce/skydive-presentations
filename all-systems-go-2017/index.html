Skydive

A real-time network topology and protocols analyzer

Why Skydive ?

  • Troubleshooting the network is hard
  • Toolbox:
    • iproute2: ip addr, ip netns, ip bridge, ip neigh, ...
    • ovs-vsctl, ovs-ofctl, ovsdb-client, ...
    • tcpdump, wireshark

Goals

  • SDN agnostic
  • Real-time / post-mortem network analysis framework
  • Lightweight, easy to deploy

Overview

  • Distributed
  • One binary to rule them all
  • Agent
    • Capture topology and flows
    • Forwards to the analyzers
  • Analyzers
    • Aggregate and store topology and flows
    • Serve API
